Anode Project¶

The purpose of this project is to create restructured text directives for annotating images. It allows you to highlight regions of an image and associate them with simple labels and longer explanations.

Basic Example¶

The is a basic example of using the anode directives.

Code¶

.. annotated-image:: 80 220 _images/shapes.png

   .. annotation:: 22 16 34 36

      **Circle**
         This circle is red. 

   .. annotation:: 10 69 59 75

      **Triangle**
         This is a yellow triangle.

   .. annotation:: 19 158 40 45

      **Square**
         And this is a square.

Result¶

Circle
This circle is red.
Triangle
This is a yellow triangle.
Square
And this is a square.

Example With Names¶

Using the directives with the additional name option.

Code¶

.. annotated-image:: 80 220 _images/shapes.png

   .. annotation:: 22 16 34 36
      :name: circle

      **Circle**
         This circle is red. 

   .. annotation:: 10 69 59 75
      :name: triangle

      **Triangle**
         This is a yellow triangle.

   .. annotation:: 19 158 40 45
      :name: square

      **Square**
         And this is a square.

Result¶

circle
triangle
square
Circle
This circle is red.
Triangle
This is a yellow triangle.
Square
And this is a square.
